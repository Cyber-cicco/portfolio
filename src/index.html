    • Présentation

    Anciennement étudiant en troisième année de double lience de philosophie et sociologie à Paris IV et développeur hobbyiste, j'ai décidé de faire un demi tour complet dans ma carrière pour faire d'une de mes passions mon métier à temps plein.
    J'ai d'abord commencé à me professionaliser via une formation en e-learning, puis voyant que le format en distanciel ne me convenait pas, j'ai parallèlement rejoins le centre de formation Diginamic via une préparation à l'emploi collective financée par l'état. Ils m'ont ensuite permis de rejoindre une formation Concepteur Développeur d'Applications en tant qu'étudiant, mais également en tant qu'alternant.
    Je suis actuellement concepteur développeur d'application pour Diginamic et Tecken. En son sein, j'ai participé à la maintenance évolutive d'une application leur servant de système d'information, j'ai créé des outils de développement pour accélerer le processus de refonte de migration du front de Angular JS à Angular, et j'ai participé à la création d'une application de suivie des stagiaires.

    Mes projets

    Spring-go

    Description :
    Utilitaire en ligne de commande de génération de code pour les stacks Spring Boot / Angular
    Permet de créer le code de base pour un certain nombre de classes Java que l'on retrouve souvent dans un projet Spring boot, comme les controllers, les dtos, les repositorys et les mappers, mais également les enums, les interfaces, les records...
    Permet de créer la structure de base d'un projet Spring Boot en définissant le nom des classes et leurs champs dans un fichier de configuration à partir d'une ligne de commande.
    Permet de créer des interfaces typescript correspondant aux données envoyées par le back en analysant le code des Dtos pour en déduire la structure des interfaces typescript
    Permet de créer des services typescript correspondant aux points d'entrées de l'application back en analysant le code des controllers du back

    Gestionnaire d'absences

    Description :
    Application web permettant à employés, managers et administrateurs de gérer les absences en entreprise.
    Permet aux employés de remplir un formulaire pour créer une demande d'absence. Le serveur vérifie ensuite certaines règles concernant ces demandes avant de l'insérer en base de données
    Permet aux employés de visualiser leurs demandes d'absences, validées ou non, ainsi que les jours fériés et les rtts employeurs
    Permet aux managers de visualiser les demandes d'absences des employés de leur département, et des les valider ou non
    Permet aux administrateurs de choisir si les jours fériés sont travaillés ou non, et des poser les RTTs employeurs
